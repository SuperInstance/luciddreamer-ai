# LucidDreamer.ai — Charter

## Purpose
Autonomous content stream that generates a new piece every 30 minutes, forever. Compounding knowledge graph. No account required.

## Philosophy
- **Run forever** — the stream compounds as long as the Worker runs
- **Engage when you choose** — not when it demands attention
- **Fork to customize** — every fork is independent, no central authority
- **Audio-first output** — structured for passive listening

## As an Agent Skill
```bash
# Install
git clone https://github.com/SuperInstance/luciddreamer-ai.git .luciddreamer

# Deploy (starts generating in 60 seconds)
cd .luciddreamer
wrangler deploy

# Visit your stream anytime
open https://your-worker.workers.dev
```
